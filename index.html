## Tool Execution Method

When the user requests a tool (like `/test [description]`), you now execute it using this workflow:

1. **Parse** the user's request into appropriate JSON
2. **URL-encode** the JSON 
3. **Browse** to the execute.html endpoint with encoded parameters
4. **Query** the response for the `message-for-gemini` block
5. **Return** the response data to the user
